# カケイボキ 機能一覧・要件定義書

## 1. アカウント管理・同期関連機能
*   **サインアップ（アカウント作成）:** メールアドレスとパスワードによる初期登録機能。
*   **ログイン / ログアウト:** Firebase Authentication を用いた認証機能。
*   **リアルタイムデータ同期:** クラウド（Firestore）とのデータ同期機能。
*   **オフライン対応:** オフライン時でもブラウザのローカルストレージ（`localStorage`）にデータを一時保存し、オンライン復帰時に再同期する機能。
*   **PWAインストール:** Webブラウザから独立したアプリとしてホーム画面/デスクトップにインストールできる機能。

## 2. 勘定科目（マスターデータ）管理機能
*   **初期科目の提供:** 現金、普通預金、給与収入、食費など、よく使われるデフォルト勘定科目の一覧設定機能。
*   **科目の独自追加:** ユーザーが必要な勘定科目を任意に追加できる機能（ID、科目名、タイプの指定）。
*   **科目の編集・削除:** 既存の勘定科目名やタイプの変更、および未使用科目の削除機能。
*   **5区分の分類:** すべての勘定科目を「資産 (Asset)」「負債 (Liability)」「純資産 (Equity)」「収益 (Income)」「費用 (Expense)」のいずれかに分類。
*   **一括リセット:** カスタム科目を破棄し、デフォルト状態に戻す機能（仕訳データは維持）。

## 3. 日々の入力（仕訳入力）機能
*   **複合仕訳対応:** 借方（Debit）と貸方（Credit）それぞれに複数行の科目を追加できる機能。
*   **貸借一致プレビュー:** 入力中の借方合計と貸方合計をリアルタイムで比較・表示する機能。
*   **入力バリデーション:** 日付・摘要の必須チェック、ならびに「借方合計 ＝ 貸方合計」が成立しない限り登録ボタンを非活性化する制御機能。

## 4. 帳簿・レポート表示機能
*   **仕訳帳 (Journal):**
    *   すべての仕訳データを時系列でリスト表示。
    *   「開始日〜終了日」による日付でのフィルタリング機能。
    *   登録済み仕訳の削除・取り消し機能。
*   **総勘定元帳 (Ledger):**
    *   特定の勘定科目（またはすべて）を選択し、その科目に関連する取引履歴（日付、摘要、相手科目なしでの借/貸方金額）を表示。
    *   取引ごとのリアルタイム残高（Running Balance）の自動計算および表示機能。
*   **貸借対照表 (Balance Sheet / B/S):**
    *   任意の基準日における「資産」「負債」「純資産」の現在残高の一覧表示。
    *   「当期純利益（収益 - 費用）」を算出し、「繰越利益剰余金」として純資産の部に自動合算するロジックを実装。
    *   左右の合計額の一致を視覚化（貸借平均の原則）。
*   **損益計算書 (Income Statement / P/L):**
    *   任意の指定期間内における「収益」と「費用」の集計表示。
    *   最終的な「当期純利益（または純損失）」を算出して表示する機能。

## 5. 定期仕訳（オートメーション）機能
*   **定期テンプレート登録:** 毎月発生する固定費（例：家賃、定期代、サブスクリプション）をテンプレート化して保存する機能。
*   **発生日設定:** 毎月何日に仕訳を起こすか（例: 25日）の指定。
*   **テンプレート有効/無効化:** 一時的に自動生成をストップするトグルスイッチ機能。
*   **一括手動生成:** 「今月分を生成」ボタンにより、条件を満たす未登録の定期仕訳を一括で実データ化。
*   **起動時自動生成:** アプリ起動（読み込み）時に当月分の定期仕訳が存在しない場合、自動的に生成を行うバックグラウンド処理。
